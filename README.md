# Zepto Product Analysis

## Overview

This project analyzes Zepto product data using **SQL** to clean the dataset, explore product trends, and generate business insights. The analysis focuses on pricing, discounts, stock availability, and product segmentation.

## Tools Used

- MySQL
- SQL

## Project Steps

- Cleaned the dataset by removing invalid records.
- Corrected price formatting using SQL.
- Performed Exploratory Data Analysis (EDA).
- Analyzed product pricing and discount percentages.
- Identified top-value products based on discounts.
- Flagged high-priced products that were out of stock.
- Segmented products by weight using CASE statements.
- Used aggregate functions, subqueries, and CTEs for analysis.

## Key Insights

- Identified the top 10 products with the highest discounts.
- Found expensive products that were unavailable.
- Compared product categories by price and discount.
- Segmented products into different weight ranges.
- Generated insights to support pricing and inventory decisions.
